Introduction Last updated: 2022-03-11 (v1.2.1)

A minimal, fast, and customizable prompt for BASH 4.0 or greater:

Minimal: No excess bloat, straight and to the point.
Customizable: Configure settings for each user, one size does not always fit all.
Hassle-free: Easy to install, easy to update (if you so chose!)

Installation

iris is able to be installed via a default install script, or through a git clone where you can choose it's working directory. Both options will require some prerequiste packages:

Prerequisites

  • git
  • sudo
  • wget (for script installation)

Optional

  • Nerd Fonts - Simply enable in your terminal (We use Hack Bold).

Installation via script

sudo su -
bash <(wget -qO - https://mschf.dev/iris)

This will install iris to /opt/iris

Installation via git

sudo su -
git clone https://github.com/mschf-dev/iris "/opt/iris"
/opt/iris/src/tools/git-install.sh

You can change the installation dir by cloning to a different directory. You will have to update the path of the git-install.sh file and run it to complete installation.

CLI Usage

iris has several cli options to help you manage the prompt.

Commands

--config [view|set] [var] [value]: view or set configuration values in ~/.config/iris/iris.conf
--defaults: resets iris configuration to default values
--disable-module [module]: disables the provided module
--enable-module [module]: enables the provided module
--help: displays iris help message
--modules: lists all installed/available modules
--reload: reloads iris configurations
--uninstall: uninstalls iris
--upgrade: upgrades iris to latest version
--version: outputs installed iris version

All configuration variables are documented below.

Configuration variables

  • prompt_input_newline - console input on new line: (true|false)
  • prompt_nerd_font - iris outputs nerd font icons: (true|false)
  • prompt_input_symbol - console input symbol
  • prompt_nerd_symbol - console input symbol for nerd font
  • prompt_username - shows current user on prompt: (true|false)
  • prompt_hostname - shows hostname on prompt: (ssh|all|none)
  • prompt_dir - shows current dir on prompt: (true|false)
  • prompt_display_error - displays error codes on prompt: (true|false)
  • prompt_seperator - seperator between prompt informations
  • prompt_wrapper - wrapper for prompt information: 2 chars max [LR]
  • prompt_success_color - changes input symbol to green if previous command returns 0: (256 color)
  • prompt_fail_color - changes input symbol to red if previous command does not return 0: (256 color)
  • prompt_user_color -sets the user color in prompt: (256 color)
  • prompt_sudo_color -sets the sudo color in prompt: (256 color)
  • prompt_info_color -sets the info color in prompt: (256 color)
  • prompt_input_color -sets the input color in prompt: (256 color)
